Apache Taverna
  • Last Published: 2015-08-14

Taverna has recently transitioned to the Apache Incubator. This website is for the Apache Taverna project and is under development.

You might want to visit the earlier taverna.org.uk website which is not affiliated with Apache.

Taverna Workflow System

Powerful, scalable, open source & domain independent tools for designing and executing workflows. Access to 3500+ resources.

Download Follow

Build scientific workflows

Taverna is an open source and domain-independent Workflow Management System – a suite of tools used to design and execute scientific workflows and aid in silico experimentation.

Taverna is moving to the Apache Incubator

Taverna has been accepted as an Apache Incubator project. In short, this means:

  • Taverna 2.5.1 will be last non-Apache release from taverna.org.uk
  • Open Development process - anyone can contribute
  • Development infrastructure hosted by apache.org
  • Taverna mailing lists have moved to @taverna.incubator.apache.org
  • Source code will move to http://git.apache.org/, with GitHub mirror

For further details, see the presentation Taverna as an Apache Incubator project and sign up to the dev@taverna.incubator.apache.org mailing list.
