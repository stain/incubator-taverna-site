Apache Taverna
  • Last Published: 2015-08-14

Apache Taverna is a incubator project within the The Apache Software Foundation (ASF).

Taverna is an open source community that’s organized by its members, including you! Whether you’re running or writing workflows, developing plugins, hosting a Taverna server, hosting a service, hacking the core, or just want to edit these pages there are opportunities for you to get in touch and ask questions, contribute or report a bug.

Incubator status

  • Taverna incubator status.

Mailing Lists

You are highly encouraged to participate on the Apache Taverna mailing lists:

Users Build/run workflows; analytical science; services Subscribe Unsubscribe Archive More info
Developers   Coding on/with Taverna; workflow/provenance science   Subscribe Unsubscribe Archive More info
Commits   Code changes; automated build reports   Subscribe Unsubscribe Archive More info

To receive messages from a list, and to be able to post to it, you need to be subscribed:

  • Send an empty message to the Subscribe address from above (any message text will be ignored)
  • Wait for the response email with a subject like *Confirm subscribe to users@taverna*
  • To confirm subscription, reply with an empty message
  • You should now be subscribed to the list

Unsubscribing is done in the same way, using the Unsubscribe address instead.

See the mailing list page for more details.

IRC chat

Some of the Apache Taverna users and developers occassionally hang out on the IRC channel #taverna.

Report or track a bug

Use the Apache Taverna JIRA issue tracker to report or track a bug.

The previous (non-Apache) Taverna JIRA can be found at: http://dev.mygrid.org.uk/issues/browse/T3

Contribute

Contribute a workflow

The best place to contribute Taverna workflows is on myExperiment.

Hosting a Taverna Server

For information on hosting a Taverna Server to run Taverna workflows, see the Server Section.

Hosting a Service for Taverna

For information on writing web services to be used in Taverna see our Web Service Developers Section.

We recommend that your services are registered in Registries such as: BioCatalogue

Contributing Code

We welcome any contributions for the source code of Apache Taverna. See how to contribute code.

Contribute a plugin

The mechanisms for writing Apache Taverna 3 plugins is still under development. Please contact the dev@taverna mailing list for details.

For Taverna 2 (non-Apache), see the Creating plugins for Taverna 2 tutorial.
