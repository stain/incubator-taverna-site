Debian/Ubuntu repository for Taverna 2.x (amd64)

Installing

    sudo apt-key adv --keyserver pgp.mit.edu --recv-key 411063A3A0FFD119
    sudo apt-add-repository "deb http://updates.taverna.org.uk/deb ./"
    sudo apt-get update
    apt-cache search taverna-| sort   # pick your edition!
    sudo apt-get install taverna-commandline-core

Notes

amd64 is the only supported platform - for i386 use the standalone downloads.
